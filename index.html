b) Responda as perguntas abaixo:

  1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?

    2. R: Disponiblizar ao usuário uma dica que descreve o valor esperado em um determinado campo de entrada.

  2. Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. R: Indica se a checkbox está inicialmente marcada quando a página é carregada.

  3. Na tag input para que serve o atributo "value"?

    4. R: Para definir o valor a ser enviado por este item. É o que a pessoa digita, nos checks e rádios, são nomes que colocamos para fazer sentido quando recebermos essa informação na outra ponta.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. R: O atributo checked é usado para indicar as checkboxes em qual rádio button devem estar ligados inicialmente se necessários.

  5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?

    6. R: Representa uma legenda para um item e uma interface de usuário. Ele pode estar associado com elemento de controle, colocando este dentro do elemento label, ou usando o atributo "for". Tal controle é chamado o controle etiquetado do elemento etiqueta.

  6. Na tag option para que serve o atributo "selected"?

    7. R: Este atributo permite especificar que um controle de formulário deve ter foco de entrada quando a página é carregada, a não ser que o usuário o substitua.

  7. Nas tags de formulário para que serve o atributo "required"?

    8. R: A tag informa ao navegador que o campo deve ser preenchido antes que o formulário possa ser enviado.

  8. Para que serve a tag textarea?

    9. R: É um elemento html que representa um controle de edição, uma caixa de texto, útil quando você quer permitir ao usuário informar um texto em formato livre.

  9. Na tag form para que serve o atributo "action"?

    10. R: Define o local em que os dados recolhidos do fomuláro devem ser enviados.

  10. Na tag form para que serve o atributo "method"?

    11. R: Serve para definir o método de envio dos dados do formulário (Get ou post)

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

    12. R: O valor do atributo da tag deve ser "post".

  12. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

    13. R: No atributo "method" na tag "form" em um fomulário que será usado para fazer o upload de arquivos, o valor deve ser "post"

  13. O que as tags fieldset e legend produzem?

    14. R: As tags "fieldset" e "legend" são usadas para criar grupos de campos em um formulário, onde "fieldset" define o grupo e "legend" fornece uma legenda ou título para esse grupo.
